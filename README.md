# 애완견 FAQ 질병 진단 챗봇 데이터셋

## 📌 프로젝트 개요
본 프로젝트는 **애완견의 이상 증상에 대해 보호자가 자주 묻는 질문(FAQ)**을 기반으로,  
질병 진단을 보조하는 챗봇 개발을 목표로 한 **QA 데이터셋 구축 프로젝트**이다.  
실제 사용자 질문 환경을 반영하기 위해 질문 표현을 다양화하고 데이터 편향을 줄이는 데 초점을 두었다.

## 🧠 사용 기술
- **Python**
- 데이터 증강(Data Augmentation)
  - 질문 문장 랜덤 변형
  - 역번역(Back Translation)

## ⚙ 구현 내용
- 기존 QA 데이터셋의 **질문 문장을 랜덤하게 변형**하여 표현 다양성 확보
- **한글 → 외국어 → 한글** 역번역 방식을 활용한 질문 증강
- 증강 과정에서 의미 왜곡을 최소화하기 위한 간단한 필터링 적용
- 챗봇 학습에 바로 활용 가능한 **질문–답변(QA) 구조 유지**

## ✍ 느낀 점
질문 데이터 증강을 통해 **데이터의 양뿐만 아니라 질문 표현의 다양성**을 확보할 수 있었다.  
특히 역번역 기법이 실제 사용자가 입력할 법한 자연스러운 질문을 생성하는 데 효과적이었다.  
이를 통해 챗봇의 일반화 성능 향상 가능성을 확인했으며,  
향후에는 질문 유형 분류나 증강 품질 자동 평가 방식까지 확장해 보고 싶다.
