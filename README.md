# 애완견 FAQ 질병 진단 챗봇 데이터셋

## 📌 프로젝트 개요

본 프로젝트는 애완견의 이상 증상에 대해 보호자가 자주 묻는 질문(FAQ)을 기반으로,
질병 진단을 보조하는 챗봇 개발을 목표로 한 QA 데이터셋 구축 프로젝트이다.
실제 사용자 질문 환경을 반영하기 위해 질문 표현을 다양화하고 데이터 편향을 줄이는 데 초점을 두었다.

## 🧠 사용 기술
- Python
- 데이터 증강(Data Augmentation)
  질문 문장 랜덤 변형
  역번역(Back Translation) 기반 증강

## 구현 내용
기존 QA 데이터셋의 질문 문장을 랜덤하게 변형하여 표현 다양성 확보
한글 → 외국어 → 한글 역번역을 활용한 질문 증강
증강 과정에서 의미 훼손 여부를 최소화하도록 필터링 로직 적용
챗봇 학습에 바로 활용 가능한 질문–답변(QA) 구조 유지

## ✍ 느낀 점

질문 데이터 증강을 통해 데이터의 양뿐만 아니라 표현 다양성까지 확보할 수 있었다.
특히 역번역 방식이 실제 사용자가 입력할 법한 자연스러운 질문 생성에 효과적이라는 점을 확인했다.
향후에는 단순 증강을 넘어 질문 유형별 분류 및 난이도 조절까지 확장해 보고 싶다.
